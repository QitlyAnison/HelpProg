Язык программирования Python

На что способен Python :

    Работа с FTP
    Работа с http запросами
    GUI (графический интерфейс)
    Создание веб-сценариев
    Программирование математических и научных вычислений
    Робототехника
    Работа с изображениями, аудио и видео файлами
    Работа с xml/html файлами
Преимущества Python среди других языков :

    Высокая сокорость выполения программ. Это связанно с тем, что основные библиотеки Python написаны на C++.
    В связи с этим вы можете писать свои собственные модули для Python на C или C++.
    В стандартныx библиотеках присутствуют средства для работы с электронной почтой, протоколами Интернета, FTP, HTTP.
    Имеет поддержку кроссплатформенности.
    Подходит для любых решений в области программирования.
